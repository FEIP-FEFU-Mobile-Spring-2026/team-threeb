## Блок 3: Детали товара

### Зачёт

Реализовать bottom sheet с деталями товара. Тап по карточке в каталоге открывает экран с полной информацией.

**Сборка**
* Проект собирается без ошибок

**Детали товара (bottom sheet)**
* Тап по товару в каталоге → открытие bottom sheet
* Отображается:
  * Изображение (`imageUrl`)
  * Название (`name`)
  * Краткое описание (`longDescription`)
  * Цена (`priceInKopecks`) — отформатирована в рублях
  * Теги (`tags`) — в виде чипов поверх картинки
* Закрытие bottom sheet (свайп вниз / кнопка / тап по затемнению)
