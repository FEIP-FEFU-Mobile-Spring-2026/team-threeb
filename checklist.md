## Блок 2: Каталог и архитектура

### Зачёт

Собрать работающий экран каталога. Пользователь видит список товаров по категориям. Данные загружаются из локального JSON-файла и хранятся в ViewModel.

**Сборка**
* Проект собирается без ошибок

**Данные**
* Данные загружаются из JSON-файла [products.json](products.json) (assets / resources)
* Категория «Новинки» формируется на клиенте: в неё попадают товары с тегом `"New"` из всех категорий
* Категория «Новинки» отображается первой в списке табов

**Каталог**
* Табы категорий — переключение фильтрует товары по выбранной категории
* Карточка товара отображает:
  * Изображение (`imageUrl`)
  * Название (`name`)
  * Цена (`priceInKopecks` — перевести в рубли)

**Навигация**
* Нижнее меню с двумя пунктами: каталог и корзина
* Корзина пока может быть пустым экраном-заглушкой

**Архитектура**
* Данные хранятся в ViewModel, не в Activity/Fragment/Composable/View
* Поворот экрана не крашит приложение и не сбрасывает данные
